움직이는 화면을 그리는
UI개발자 김경화 입니다.

이력서 바로가기

저는 동적 웹사이트 및 사용자가 만족할수 있는 서비스를 만드는 것에 열정을 가진 3년차 웹 퍼블리셔 입니다.

시멘틱한 마크업, 컴포넌트 단위의 스타일시트 작성, 자바스크립트로 인터렉션을 구현하여,
화면정의서와 디자인 시안을 완벽하게 웹 페이지로 전환할 수 있습니다.

저는 복잡한 문제를 파악하고 해결하는 재능이 있고, 제 기술과 지식을 지속적으로 향상시키는 추진력이 있습니다.

작업의 효율성과 완성도를 높이기 위해 디자인 팀과 기획 팀 모두와 협력하는 데 탁월하며,
직관적이고 효율적인 웹 경험을 만들기 위해 늘 사용자의 요구를 이해하고 싶습니다.

저는 평생 배우는 사람이고 제 기술을 향상시킬 새롭고 혁신적인 방법을 항상 찾고 있습니다.
꾸준히 성장하고 발전하며 함께 일하고싶은 동료가 되기를 바랍니다.

이렇게 일합니다.

  • 아이콘-연필

    Structure

    • 마크업에 앞서 CSS 폴더트리를 세분화하여 화면 구조를 설계합니다.
    • 목적에 맞는 레이아웃과 컴포넌트로 단위로 디자인을 분리하여 재사용성을 높입니다.
    • 사용되는 태그의 의미를 고려한 Semantic Markup 코드를 작성합니다.
  • 아이콘-키보드

    Efficiently work

    • 작업속도 향상을 위해 CSS전처리기(Sass)를 도입, BEM CSS 방법론을 사용합니다.
    • 유지보수가 쉽도록 클린코드를 작성합니다.
  • 아이콘-유저

    Easy to use

    • 웹표준을 준수하고 웹접근성을 이해하여 누구나 사용할 수 있는 화면을 구성합니다.
    • 반응형 웹 제작과 크로스 브라우징으로 다양한 환경에서도 최적화 되도록 구현합니다.
    • 사용자 입장에서 시나리오를 작성하고 테스트 하여, 쉽고 직관적인 사용자 경험을 이끌어냅니다.
  • 아이콘-말풍선

    Communication

    • 기획자와 디자이너, 개발자와 항상 소통하고 지식을 공유하며 더 나은방향을 고민합니다.
    • Git, jira, confluence 등의 협업툴들 사용하여 구성원과 협업합니다.

이렇게 작업했어요.
